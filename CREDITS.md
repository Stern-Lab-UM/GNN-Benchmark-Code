# Credits

This repository accompanies the preprint:

Krajnc, M., Comi, T., Miao, S., Hafeez, A., Serviansky, H., Li, P.,
and Stern, T. (2026). *A Controlled in Silico Benchmark for GNN
Prediction of Tissue Dynamics*. Preprint.

Online preprint URL/DOI: pending public posting.

## Code Contributions

- Siqi Miao and Pan Li provided the MPNN code used for GraphSAGE, GAT,
  GIN, and PNA.
- Troy Comi provided the PPGN code.
- Tomer Stern adapted the GNN code for the updated manuscript analyses and
  ablation analyses.
- Matej Krajnc provided the tissue-simulation and spring-embedding code.
- Adnan Hafeez supported code installation and computing-environment setup.
